# サポートされているプラットフォーム

このトピックでは、Rights Management サービス SDK 2.1 でサポートされているクライアントとサーバー プラットフォームを識別します。

## サポートされているプラットフォーム

-   Windows 10
-   Windows Server 2012 R2
-   Windows Server 2012
-   Windows 8
-   Windows Server 2008 R2 SP1
-   Windows 7 + SP1

**注**: テンプレート機能の使用は Windows Server 2008 以降でサポートされます。
