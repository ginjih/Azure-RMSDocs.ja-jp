# <a name="azure-information-protection-documentation"></a>Azure Information Protection のドキュメント

社外で共有する電子メール、ドキュメント、および機密データを制御し、セキュリティで保護します。 Azure Information Protection での簡単な分類から埋め込みラベルとアクセス許可、データ保護の強化は常時提供され、保存場所、共有先は関係ありません。
